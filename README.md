# ORDA Smart Event System

Веб-платформа для регистрации и управления конференциями и мероприятиями.

## Запуск

```powershell
cd C:\Users\adilb\OneDrive\Desktop\Diplom
node server.js
```

Открыть сайт:

```text
http://localhost:3000
```

## Настройка MongoDB

Проект использует MongoDB Atlas. Переменные находятся в `.env`:

```text
PORT=3000
JWT_SECRET=...
MONGODB_URI=...
```

Если `/health` показывает:

```json
{ "database": "connecting" }
```

и в консоли появляется ошибка про MongoDB Atlas IP whitelist, нужно:

1. Открыть MongoDB Atlas.
2. Перейти в `Network Access`.
3. Добавить текущий IP-адрес компьютера.
4. Подождать 1-2 минуты.
5. Перезапустить сервер.

## Основной функционал

- Регистрация и вход пользователя.
- JWT авторизация.
- Роли `admin` и `user`.
- Просмотр мероприятий.
- Поиск и фильтрация мероприятий.
- Онлайн, офлайн и гибридные мероприятия.
- Регистрация участника на мероприятие.
- Личный кабинет с моими регистрациями.
- Создание, редактирование и удаление мероприятий админом.
- Просмотр заявок участников.
- Check-in участников.
- Dashboard статистики.
- Три языка: русский, қазақша, English.
- Light/Dark UI.

## Быстрый сценарий демонстрации

1. Запустить проект командой `node server.js`.
2. Открыть `http://localhost:3000`.
3. Зарегистрировать первого пользователя. Первый пользователь автоматически получает роль `admin`.
4. Создать мероприятие в Admin Dashboard.
5. Выйти и зарегистрировать обычного пользователя.
6. Найти мероприятие, открыть детали и зарегистрироваться.
7. Открыть личный кабинет и показать мои регистрации.
8. Вернуться под админом и показать заявки, статистику и check-in.
